** 关于 JavaWebKit ** 
--------------------
本套源码提供了 java web 开发需要的一套基础功能。  

**主要技术** 
--------------------
* spring mvc + jpa(hibernate) + log4j + junit + sitemesh
* 开发工具： IDEA
* 依赖库管理：maven
